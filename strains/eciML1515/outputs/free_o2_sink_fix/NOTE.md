# Closing the uncosted O2-sink reactions (Parsa's audit)

**Model-quality correction, default ON** (`close_free_o2_sinks: true` in
`configs/defaults.yaml`; reversible). Applied in `providers.close_free_energy_sinks()`
after the ecModel loads; the closed reaction IDs are recorded in each run's
`resolved_config.yaml`. Catalase (`CATNo1`) and superoxide dismutase (`SPODMNo1`) are
real enzymes and are left **open**.

## The four reactions closed (eciML1515)

| reaction | stoichiometry | enzyme cost |
|----------|---------------|-------------|
| `QMO2No1` | 2 O2 + ubiquinol → 2 superoxide + ubiquinone | ~free (prot_P0ADU2, 1.07e-8) |
| `QMO3No1` | 2 O2 + menaquinol → 2 superoxide + menaquinone | ~free (same enzyme) |
| `MOX` | malate + O2 → H2O2 + oxaloacetate | none (no enzyme-cost entry) |
| `CU1Opp` | 4 Cu⁺ + 4 H⁺ + O2 → 4 Cu²⁺ + 2 H₂O | none |

All four are forward-only (lb 0, ub 1000 → ub 0); no `_REV` variants exist. Together they
consume O2 at ~zero enzyme cost and short-circuit the electron transport chain.

## Growth / TPC impact — negligible (tuned model, rich BHI, v3 posterior medians)

| descriptor | without closure | with closure | Δ |
|------------|-----------------|--------------|---|
| r_max (h⁻¹) | 2.1609 | 2.1558 | **−0.24 %** |
| T_opt (°C) | 39.0 | 39.0 | 0.0 |
| E_a (eV) | 0.9112 | 0.9095 | −0.002 |
| CT_max (°C) | 45.969 | 45.969 | −0.0001 |

Growth falls **−0.235 %** (≈ Parsa's ~−0.3 %); the TPC shape/position is unchanged. This
shift is far inside the P2 v3 posterior width (σ_disc ≈ 0.20 h⁻¹; σ posterior CI
[0.67, 0.99]), so **the Bayesian calibration is not re-run** — the tuned posterior is
unaffected within uncertainty.

## O2 audit at the optimal T (39 °C)

| quantity | without closure | with closure |
|----------|-----------------|--------------|
| O2 uptake (exchange `EX_o2_e_REV`) | 49.53 | 1.31 |
| gross metabolic O2 consumption | 198.10 | 1.31 |
| gross / uptake (futile-cycle ratio) | **4.0** (75 % futile) | **1.0** (net = uptake) |
| QMO2+QMO3 flux | 99.05 | 0 |
| terminal oxidase flux | CYTBO3 0 / CYTBD 0 | CYTBD (`CYTBD2ppNo1`) 2.61 |

**Before:** 75 % of gross O2 turnover was a futile cycle (QMO2/QMO3 consume O2 → superoxide,
regenerated by SOD/catalase), and **no** flux went through a genuine terminal oxidase.
**After:** the futile cycle is gone, net intracellular O2 consumption equals the O2 uptake
(1.31 = 1.31), and O2 routes through a real, enzyme-costed terminal oxidase — cytochrome
**bd** (`CYTBD2pp`); the model selects bd over bo3 (CYTBO3) at this low respiratory O2 demand.

## Emergent Van Derlinden validation on the corrected model — matches the report

r_max 1.03, T_opt 40.0, E_a 0.65, CT_max 51.7 — within rounding of the report's a-priori
rich values (r_max ≈ 1.04, T_opt ≈ 40, E_a ≈ 0.63). The growth results are essentially
identical.

## Scope

Permanent default correction only. The full report and the heavy control/decomposition
runs are **not** rebuilt here; the downstream analyses (the Eₐ dissection, and any refresh
of the report's control/identifiability section — where QMO2/QMO3 currently appear) will
now inherit the corrected model automatically.
